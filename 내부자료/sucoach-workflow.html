전체 접근 권한
업무 프로세스
1
초기 설정
학생 계정 등록
신규 학생의 기본 정보(이름, 이메일, 학교, 학년, 유형)를 시스템에 등록하고 계정을 생성합니다. 학부모 계정도 함께 생성합니다.
users 테이블 INSERT 학생/학부모 계정 쌍 생성 SHA-256 비밀번호 처리
2
접수
수행평가 접수 등록
학생이 요청한 수행평가를 시스템에 등록합니다. 과목, 마감일, 상세 내용, 담당 강사를 지정하고 초기 단계를 '접수'로 설정합니다.
assignments 테이블 INSERT tutor_id 배정 status = '접수'
3
모니터링
전체 진행 현황 관리
모든 학생의 수행평가 진행 단계를 대시보드에서 실시간으로 확인합니다. 지연 건을 파악하고, 필요 시 담당 강사를 변경하거나 마감일을 조정합니다.
전체 student 뷰 단계별 필터링 강사 재배정
4
파일 관리
업로드 파일 전체 열람
각 수행평가에 업로드된 모든 파일(분석 자료, 초안, 피드백, 최종본)을 열람하고 관리합니다. Storage 경로에 직접 접근 가능합니다.
Supabase Storage 접근 files 테이블 전체 조회
5
완료
결과 데이터 확인 및 정산
완료 처리된 수행평가의 최종 점수와 등급을 확인합니다. 학생별 성과를 집계하고 프로그램 효과를 분석합니다.
result_score / result_grade 학생별 통계 기간별 리포트
👨‍🏫
강사 · 재우
수행평가 전략 코칭의 핵심 역할. 분석→면담→초안→피드백→수정→최종까지 8단계 워크플로우를 주도합니다.

Stage 4

Stage 5
피드백완료
Stage 6
수정중
Stage 7
최종확인
Stage 8
완료
1
접수
배정 수행평가 확인
Admin이 배정한 수행평가 내용을 확인합니다. 과목, 마감일, 학생 정보, 수행평가 세부 요건을 파악하고 코칭 계획을 수립합니다.
수행평가 목록 조회 학생 정보 확인 요건 검토
2
분석완료
수행평가 요건 분석 + 방향 설정
채점 기준, 루브릭, 출제 의도를 분석합니다. 학생의 현재 수준을 진단하고 최적의 전략 방향을 설정합니다. 분석 자료 파일을 업로드하고 단계를 진행합니다.
파일 업로드 (분석자료) status → '분석완료'
3
면담완료
학생 면담 진행
학생과 1:1 면담을 통해 학생의 생각, 아이디어, 방향성을 파악합니다. 면담 내용을 바탕으로 초안 구성을 확정하고 단계를 업데이트합니다.
면담 노트 업로드 status → '면담완료'
4
초안대기
초안 작성 및 업로드
면담 내용을 바탕으로 수행평가 초안을 작성합니다. 완성된 초안 파일을 시스템에 업로드하고 학생/학부모가 확인할 수 있도록 공개합니다.
파일 업로드 (초안) status → '초안대기' 학생 열람 가능
5
피드백완료
피드백 수렴 및 반영 확인
학생/학부모로부터 받은 피드백 내용을 검토합니다. 반영 가능한 사항과 방향성을 정리하고 수정 계획을 수립합니다.
피드백 내용 확인 status → '피드백완료'
6
수정중
초안 수정 작업 진행
피드백을 반영하여 수행평가를 수정합니다. 내용의 완성도를 높이고, 채점 기준에 맞게 조정합니다. 수정 과정 파일을 필요 시 업로드합니다.
파일 업로드 (수정본) status → '수정중'
7
최종확인
최종본 업로드 및 제출
수정이 완료된 최종본을 업로드합니다. 학생/학부모가 최종본을 확인하고 제출 승인을 합니다. 이상이 없으면 완료 단계로 진행합니다.
파일 업로드 (최종본) status → '최종확인' 학생 최종 검토
8
완료
결과 입력 — 점수 / 등급 기록
수행평가 제출 후 받은 실제 점수와 등급을 시스템에 입력합니다. 학생과 학부모가 결과를 확인할 수 있으며, 코칭 효과를 분석하는 데 활용됩니다.
result_score 입력 result_grade 입력 status → '완료'
🎒
학생
수행평가 코칭의 주인공. 진행 현황을 확인하고, 파일을 수령하며, 피드백을 제공하고 최종 결과를 확인합니다.
예) jisu@sucoach.kr
student-page.html
본인 수행평가만 접근
이용 흐름
1
로그인
학생 계정으로 접속
이메일 + 비밀번호로 로그인합니다. 시스템이 역할을 자동으로 확인하여 학생 전용 페이지로 이동합니다.
SHA-256 인증 localStorage sc_user 저장 student-page.html 이동
2
현황 확인
수행평가 진행 단계 확인
현재 진행 중인 수행평가 목록과 각 수행평가의 단계(접수/분석완료/면담완료/초안대기…)를 카드 형태로 확인합니다.
assignments 목록 조회 단계별 배지 표시
3
파일 수령
분석자료 / 초안 / 최종본 다운로드
강사가 업로드한 파일(분석 자료, 초안, 최종본)을 다운로드합니다. 파일은 Supabase Storage에서 공개 URL로 제공됩니다.
파일 목록 조회 공개 URL 다운로드 다운로드 가능: 초안·최종본·분석자료
4
피드백
수정 의견 / 피드백 파일 업로드
초안을 검토한 후 수정 요청 사항이나 의견을 파일로 업로드하여 강사에게 전달합니다.
파일 업로드 (학생 피드백) uploader_role = 'student'
5
최종 검토
최종본 확인 및 승인
강사가 수정 완료한 최종본을 다운로드하여 검토합니다. 이상이 없으면 강사에게 완료 승인을 전달합니다.
최종본 다운로드 내용 최종 확인
6
결과 확인
최종 점수 및 등급 확인
수행평가 제출 후 나온 실제 결과(점수, 등급)를 시스템에서 확인합니다. 강사가 입력한 결과가 표시됩니다.
result_score 열람 result_grade 열람
👨‍👩‍👧
학부모
자녀의 수행평가 진행 상황을 모니터링합니다. 읽기 전용 접근이 기본이며, 진행 단계·파일·결과를 실시간으로 확인할 수 있습니다.
예) jisu.parent@sucoach.kr
parent-page.html
자녀 데이터 읽기 전용
이용 흐름
1
로그인
학부모 계정으로 접속
학부모 전용 이메일과 비밀번호로 로그인합니다. 시스템이 자동으로 자녀의 student_id와 연결하여 학부모 전용 대시보드로 이동합니다.
SHA-256 인증 linked_student_id 연결 parent-page.html 이동
2
현황 모니터링
자녀 수행평가 진행 단계 확인
자녀가 현재 진행 중인 모든 수행평가의 단계를 실시간으로 확인합니다. 언제, 어떤 단계에 있는지 투명하게 파악할 수 있습니다.
자녀 assignments 조회 단계별 현황 표시 마감일 확인
3
파일 열람
업로드된 파일 확인
강사가 업로드한 분석 자료, 초안, 최종본 파일을 다운로드하여 내용을 직접 확인합니다. 자녀의 수행평가 방향성과 내용을 파악할 수 있습니다.
파일 목록 열람 공개 URL 다운로드
4
결과 확인
최종 점수 및 등급 열람
수행평가가 완료되면 강사가 입력한 실제 점수와 등급을 확인합니다. 자녀의 수행평가 성과를 한눈에 파악할 수 있습니다.
result_score 확인 result_grade 확인 읽기 전용
